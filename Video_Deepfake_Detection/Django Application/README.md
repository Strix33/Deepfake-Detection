# Django Deepfake Detection Application

This is the central web application dashboard that coordinates video upload, face tracking, and model predictions.

## 🚀 Setup Instructions

### Prerequisites
Ensure you have Python 3.8+ installed.

### Setup and Running
1. Navigate to this directory:
   ```bash
   cd "Video_Deepfake_Detection/Django Application"
   ```
2. Install Python packages:
   ```bash
   pip install -r requirements.txt
   ```
3. Run database migrations:
   ```bash
   python manage.py migrate
   ```
4. Start the Django application server:
   ```bash
   python manage.py runserver
   ```
5. Open your browser and go to:
   **http://127.0.0.1:8000/**
